Bunny Hopper

Bunnyhopper is a 2D multiplayer platform game in which players can compete against one another at several different levels. The players can use their mobile phones as game controllers.

No native app is needed for the controller, as it can be opened directly in the smartphone browser. The player’s keystrokes are sent to the STB via MAF Rooms. The game runs entirely in Canvas (in a WebGL context), which uses a WebGL framework developed in-house.

Techniques used

  • JavaScript
  • MAF Rooms (WebSockets)
  • Canvas (WebGL context)
